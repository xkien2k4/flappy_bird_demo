Flappy Bird Demo

Flappy Bird Demo là một bản demo trò chơi Flappy Bird đơn giản viết bằng Python. Mục tiêu của dự án là cung cấp một ví dụ cơ bản về cách lập trình game 2D với Python — bạn có thể sử dụng repo này để học hoặc mở rộng thêm tính năng.

🚀 Yêu cầu

Python ≥ 3.x

Cài thư viện 
pip install pygame 

🕹️ Hướng dẫn chơi

Nhấn Space (hoặc phím tương đương) để cho chim “flap” và bay lên.

Tránh chạm vào các cột phía trước.

Mỗi lần vượt qua cột, bạn nhận điểm.

Game kết thúc khi chim chạm đất hoặc va vào cột.
